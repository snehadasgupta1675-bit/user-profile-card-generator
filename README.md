# User Profile Card Generator

## 📌 Project Description
This is a full stack web application that generates user profile cards using HTML, CSS, JavaScript, Node.js, and Express.js.

## 🚀 Features
- User input form
- Dynamic profile card generation
- Backend API using Express.js
- Image support
- Responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## 📷 How it works
1. User enters details in form
2. Data is sent to backend
3. Backend returns response
4. Profile card is displayed on UI
